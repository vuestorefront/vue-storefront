# ElasticSearch data formats

The service is using ElasticSearch data format compliant with ElasticSuite for Magento 1.x/2.x from Smile.

# Product type

The product data format is combined form of the following Magento2 REST API calls:

  • catalogProductRepositoryV1GetListGet
  • catalogInventoryStockRegistryV1GetStockItemBySkuGet
  • configurableProductLinkManagementV1GetChildrenGet
  • configurableProductOptionRepositoryV1GetListGet
